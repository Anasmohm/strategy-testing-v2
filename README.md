# منصة ادارة الاستثمار

تحول هذا المشروع من اختبار محفظة واحدة الى هيكل مؤسسة استثمارية قابلة لادارة عدة محافظ. المحفظة الحالية هي اول محفظة معتمدة داخل المؤسسة، وليست كل المؤسسة.

## الهيكل الاداري

```text
مجلس الادارة
  المدير التنفيذي
    ادارة البيانات
    ادارة التحليل والاستثمار
    ادارة الحلول
    ادارة الصناديق والمحافظ
      محفظة النمو الأمريكية الشرعية
```

## المجلدات

| المجلد | الدور |
|---|---|
| `corporate` | الحوكمة، سجل الاعتمادات، وسجل المحافظ لدى المدير التنفيذي |
| `departments/data_management` | ملكية بيانات السوق وجودتها وتحديثها |
| `departments/investment_research` | التشخيص وتصميم الاستراتيجيات والاختبارات |
| `departments/solutions` | الداشبورد والتشغيل والنشر والتحديث الالي |
| `departments/portfolio_management` | ادارة المحافظ المستقلة |
| `portfolios/sharia_us_growth_v1` | ملفات وتقارير المحفظة الرسمية الحالية |

## المحفظة الحالية

المحفظة النشطة هي `sharia_us_growth_v1` باسم **محفظة النمو الأمريكية الشرعية**:

- الاسهم: `AMD`, `ANET`, `AVGO`, `LRCX`, `MRVL`, `NVDA`, `PANW`, `SHOP`, `TSLA`.
- بداية العرض الرسمي: `2021-01-01`.
- رأس المال الابتدائي: `$10,000`.
- التنفيذ: بيانات `EODHD` بفاصل خمس دقائق.
- الوقف المتحرك: يحتسب من اعلى يوم مكتمل بدرجة `0.5%`، ويتفعل في جلسة التداول التالية.

## التشغيل والتوافق

نقاط التشغيل الحالية في جذر المشروع باقية عمدا حتى لا تنقطع الروابط او مهام التحديث:

- `paper_portfolio_v31.py`: يبني المحفظة الرسمية.
- `settings_server_v2.py`: يخدم اعدادات الداشبورد المحلي.
- `update_github_pages.py`: يبني وينشر التحديث اليومي.
- `reports/portfolio_dashboard.html`: رابط الداشبورد المتوافق حاليا.

بعد كل بناء رسمي يزامن النظام التقارير الرسمية الى:

`portfolios/sharia_us_growth_v1/reports`

وبذلك يمكن اضافة محافظ اخرى لاحقا دون خلط تقاريرها او سياساتها مع المحفظة الحالية.
هذه النسخ المنظمة مخرجات مولدة محليا ولا تضاف الى مستودع النشر حتى لا نكرر ملفات التقرير الكبيرة مرتين في كل تحديث.

## الفصل بين الرسمي والتجريبي

- صفحات `portfolio_*` هي التقارير الرسمية للمحفظة النشطة.
- ملفات `experimental_*` وملفات المقارنة هي ارشيف بحثي داخلي ولا تظهر في واجهة المحفظة الرسمية.
- الاعتماد الاستثماري يسجل في `corporate/board/approval_register.md`.

## تنبيه منهجي

هذه محاكاة لمحفظة ورقية وليست تنفيذا لاموال حقيقية. استعمل عام `2021` لاختيار درجة الوقف المعتمدة، لذا تعد النتائج من بداية `2022` القراءة الاكثر تحفظا خارج فترة اختيار سياسة الوقف.
