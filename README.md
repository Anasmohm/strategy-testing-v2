# اختبار الاستراتيجيات - النسخة الثانية

هذه النسخة تختلف عن النسخة الأولى.

الهدف هنا ليس اختيار أفضل مؤشر جاهز من قائمة مؤشرات، بل:

1. تشخيص سلوك كل سهم تاريخيًا.
2. فهم طبيعة السهم: اختراق، ترند، ارتداد، تذبذب، حساسية أخبار، سرعة تحقيق الربح.
3. تصميم استراتيجية مخصصة لكل سهم بناءً على التشخيص.
4. اختبار الاستراتيجية المصممة للتحقق، لا لاختراعها عشوائيًا.
5. تشغيل محفظة ورقية وداشبورد متابعة وتحليلات.

## الفرق عن النسخة الأولى

النسخة الأولى:
- تختبر قوالب جاهزة مثل RSI وBreakout وSMA وMACD.
- تختار القواعد التي حققت شروطًا تاريخية.

النسخة الثانية:
- تبدأ بتقرير تشخيص للسهم.
- تصمم استراتيجية خاصة بالسهم.
- تستخدم الاختبار للتحقق من جودة التصميم.

## المخرجات المخطط لها

- تقرير تشخيص لكل سهم.
- استراتيجية مصممة لكل سهم.
- باك تست للتحقق من الاستراتيجية.
- داشبورد صفقات ورقية.
- صفحة تحليلات تقارن جودة التصميمات.
