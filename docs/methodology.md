# منهج النسخة الثانية

## 1. تشخيص السهم

لكل سهم نحسب ونلخص:

- قوة الترند.
- سلوك الاختراق.
- سلوك الارتداد بعد الهبوط.
- التذبذب اليومي.
- سرعة تحقيق الربح.
- أفضل مدة احتفاظ تاريخيًا.
- حساسية الأخبار والأرباح.
- حجم التداول والسيولة.
- نوع الخسائر: وقف، فجوات، تذبذب، فشل اختراق.

## 2. تصميم الاستراتيجية

بعد التشخيص نختار تصميمًا مناسبًا:

- استراتيجية اختراق إذا كان السهم يحترم القمم.
- استراتيجية ارتداد إذا كان السهم يرتد من الهبوط.
- استراتيجية ترند إذا كان يحترم المتوسطات.
- استراتيجية تجنب إذا كان السهم عشوائيًا أو عالي الفجوات.

## 3. التحقق

نختبر التصميم على بيانات تاريخية:

- نسبة الفوز.
- متوسط الربح.
- متوسط الخسارة.
- السحب.
- عدد الصفقات.
- حساسية المعاملات.

## 4. التشغيل الورقي

نفتح صفقات ورقية فقط عندما تتحقق شروط الاستراتيجية المصممة للسهم.
