# الطبقة المؤسسية

هذا المجلد يمثل مستوى المؤسسة، وليس محفظة بعينها.

## التسلسل الإداري

1. مجلس الادارة: يعتمد التوجه العام والمحافظ والسياسات الجوهرية.
2. المدير التنفيذي: ينسق عمل الادارات ويرفع قرارات الاعتماد.
3. الادارات التنفيذية: البيانات، التحليل والاستثمار، الحلول، والصناديق والمحافظ.

## قواعد الحوكمة

- كل محفظة لها تفويض وسياسة مخاطر وملفات نتائج مستقلة.
- نتائج الاختبارات والمقارنات لا تصبح سياسة رسمية إلا بعد اعتمادها.
- مجلد `reports` في جذر المشروع باق كنقطة تشغيل متوافقة للداشبورد الحالي والنشر.
- النسخة المنظمة لكل محفظة تحفظ داخل `portfolios/<portfolio_id>/reports`.

## الملفات

- `organization.json`: تعريف المؤسسة والادارات.
- `executive_office/portfolio_registry.json`: سجل المحافظ وحالتها.
- `board/approval_register.md`: قرارات الاعتماد الرئيسية.
