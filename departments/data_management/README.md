# ادارة البيانات

## المسؤولية

تملك هذه الادارة مصدر البيانات، التحديث، التغطية التاريخية، وفحوص جودة جلسات التداول.

## الانظمة الحالية

| الوظيفة | الملف التشغيلي الحالي |
|---|---|
| مصدر البيانات الرسمي وتجميع البيانات | `eodhd_official_data.py` |
| سحب بيانات الخمس دقائق والتحقق التجريبي | `download_eodhd_intraday_experiment.py` |
| اصلاح جلسات البيانات الناقصة | `repair_eodhd_5m_experiment.py` |
| التحقق من مصدر التنفيذ | `validate_eodhd_intraday.py` |

## مصدر البيانات المعتمد

- المصدر: `EODHD EOD-IntraDay All World`.
- بيانات التنفيذ: شموع خمس دقائق للجلسة النظامية.
- بيانات تهيئة المؤشرات اليومية: تبدأ من عام `2020`.
- بيانات التنفيذ للمحفظة الحالية: تبدأ من اول جلسة تداول في عام `2021`.

هذه الادارة تخدم كل المحافظ المستقبلية؛ بيانات السوق ليست ملكا لمحفظة واحدة.
