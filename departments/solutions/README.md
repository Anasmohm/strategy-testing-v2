# ادارة الحلول

## المسؤولية

تبني وتشغل الادوات التي تستخدمها الادارات الاخرى: الداشبورد، صفحات القرار، الخادم المحلي، التحديث الالي، والنشر.

## الانظمة الحالية

| الخدمة | الملف التشغيلي الحالي |
|---|---|
| بناء الداشبورد الرسمي | `paper_portfolio_v31.py` |
| خادم الاعدادات المحلي | `settings_server_v2.py` |
| تشغيل الداشبورد المحلي | `start_dashboard_v2.ps1` |
| اعداد صفحات النشر | `prepare_github_pages.py` |
| تحديث ونشر GitHub Pages | `update_github_pages.py` |
| الجدولة اليومية | `.github/workflows/daily-dashboard.yml` |

## مبدأ التشغيل

تظل نقاط التشغيل الحالية في جذر المشروع للتوافق مع الخادم المحلي والتحديث اليومي، بينما تنسخ مخرجات كل محفظة الى مجلدها المستقل بعد كل بناء.
