# محفظة النمو الأمريكية الشرعية

## التعريف

هذه اول محفظة تحت ادارة الصناديق والمحافظ. هي محفظة ورقية لا تنفذ اوامر مالية حقيقية.

## ملفات المحفظة

- `mandate.json`: غرض المحفظة والكون الاستثماري.
- `risk_policy.json`: سياسة المخاطر والتنفيذ والخروج.
- `run_portfolio.py`: تشغيل بناء تقارير هذه المحفظة عبر المحرك المعتمد.
- `reports`: التقارير والبيانات الناتجة لهذه المحفظة فقط، ويتم تحديثها بعد كل بناء رسمي.
- `status/latest_portfolio_status.json`: اخر ملخص تشغيلي للمحفظة.

## رابط التشغيل المتوافق

يستمر الداشبورد المحلي الرئيسي بالعمل من:

`http://127.0.0.1:8782/reports/portfolio_dashboard.html`

أما النسخة المنظمة داخل المحفظة فتكون:

`http://127.0.0.1:8782/portfolios/sharia_us_growth_v1/reports/portfolio_dashboard.html`
