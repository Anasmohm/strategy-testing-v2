# المحافظ

هذا المجلد هو سجل المحافظ التي تديرها المؤسسة. كل مجلد داخله يمثل محفظة مستقلة لها تفويض ومخاطر وتقارير منفصلة.

## المحفظة النشطة

- `sharia_us_growth_v1`: محفظة النمو الأمريكية الشرعية.

يبقى `reports/portfolio_dashboard.html` في جذر المشروع رابطا متوافقا للتشغيل الحالي، بينما النسخة المنظمة للمحفظة تحفظ في `portfolios/sharia_us_growth_v1/reports`.
